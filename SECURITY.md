# Security policy

## Supported versions

Security fixes are provided for the latest published minor release.

## Reporting a vulnerability

Do not open a public issue for an undisclosed vulnerability. Use GitHub's
private vulnerability reporting for
[`agporto/tiny3d-rs`](https://github.com/agporto/tiny3d-rs/security/advisories/new).

Include affected versions, platform details, reproduction steps, impact, and
any proposed mitigation. You should receive an acknowledgement within seven
days. Please allow time for investigation and coordinated disclosure.
